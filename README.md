# Product_Management

Spring Boot React Crud Example Technologies

Backend : Spring Boot ,Rest Api

Frontend â€“ React (React Hook, React Router 6.0 )

Build

1) Build Server Side
   
     npx create-react-app product_management_frontend

3) Build and run client side
   
   $ cd product_ui
   $ npm start
